2D-Visualisierung des Schulgebäudes

HTL Leonding Raumvisualisierung

Übersicht

  • Interaktiver 2D-Grundriss der HTL Leonding
  • Implementiert als statische SVG-Komponente
  • Angular-basierte Interaktivität

Technische Umsetzung

SVG-Implementierung

  • Hardcodierte SVG-Definitionen im HTML
  • Separate SVGs pro Stockwerk
  • ViewBox-basierte Skalierung

Raumdarstellung

  • Rechteckige Räume: <rect> Elemente
  • Komplexe Formen: <polygon> Elemente
  • Strukturierte Gruppierung mit <g> Tags

Angular Integration

  • ngSwitch für Stockwerksauswahl
  • Event Emitter für Rauminteraktionen
  • Dynamische CSS-Klassen für Hervorhebungen

Interaktive Features

  • svg-pan-zoom Bibliothek Integration
  • CSS-basierte Hover-Effekte
  • Click-Events für Raumauswahl

Code-Struktur


<div class="svg-viewer">
  <span [ngSwitch]="displayArea?.name">
    <svg *ngSwitchCase="'firstfloor'">
      <g class="room-box">
        <rect id="room1" (click)="emitSection($event)">
      </g>
    </svg>
  </span>
</div>

Komponentenstruktur


@Component({
  selector: 'app-svg-viewer'
})
export class SvgViewerComponent {
  @Input() displayArea: Area;
  @Output() sectionClicked = new EventEmitter();
}

  • Hover-Highlights für Räume
  • Dynamische Selektionshervorhebung

  • Statische SVG-Definition
